``/csclass delete <职业名称>``

``/csclass list``

## 游戏模式
### 团队
达到配置里的 最高人数 

### 感染（生化）
 2种阵营 A阵营 打死 B阵营的人 B为A的人  B死完 A赢   B打死 所有A A赢 A只有 2跳明  A差不多就是僵尸  B就人类
 
### 守卫
 地图中有个xx血的怪 谁先打死 谁赢。
（类似于起床战争，家里有个怪。击杀后。那方无法复活！）
